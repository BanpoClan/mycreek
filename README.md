# mycreek
这是一套框架
基于ABP3.4做的一个demo
网上很多资料感觉都比较老。DTO  ENTITY  的一些类都被移除了。目前为止，最新的是3.5。若是想学ABP实际的demo，真心建议使用这个。

这个demo除了ABP框架自带的，包含了以下内容：
1.集成了ueditor插件，用来自定义表单（需要修改ueditor.all.js，不然编辑器内不能插入控件，即便插入了也获取不到对应的html代码）
2.在界面添加字段然后刷新到数据库
3.从数据库里面构造菜单
4.集成了ztree bootstrap 的树形控件
5.集成了jtable表格控件
